# Tournament Prep Skill

## Overview
Generates pre-tournament preparation packages for the coaching staff. Combines player profiling, course strategy, logistics planning, and a faith-grounded pregame message into a single actionable document.

## Usage
```
autonomaton> tournament prep
autonomaton> match prep for cathedral
autonomaton> prepare for match
```

## What It Does
1. Reads the upcoming match from seasonal context
2. Pulls player profiles and identifies lineup candidates
3. Generates scouting cards with player-specific coaching cues
4. Creates course strategy notes for decision holes
5. Writes a Pregame Word for Coach to share with the team
6. Produces a logistics checklist

## Zone
Green — read-only analysis and generation. No external actions.

## Output
Structured tournament package saved to `output/reports/`. Can be printed or shared with assistant coaches.

## Telemetry Value
Every tournament prep package generates data about lineup decisions, player assessment accuracy, and course strategy — feeding the Ratchet's understanding of coaching patterns over time.
